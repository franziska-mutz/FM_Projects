## Overview:  
## Project Details: 
### Objectives: 
* Data Exploration

### Technical Environment:
* SQLite using Joins etc. 
* Editing platform: SQLiteStudio 
* Database locally on personal machine 
### Data Sources/Sets:
* Production date for several agricultural products based on the United States Department of Agriculture (USDA) data reports
 

## Project Breakdown: 
* **Business/Use Case Problem**
    Generating insights from this data to aid in future planning and decision-making. Queries provide data for reports, strategic discussion and meetings by
    * Assessing state-by-state production for each commodity.
    * Identifying trends or anomalies.
    * Offering data-backed suggestions for areas that may need more attention
    
* **Data Collection**
  * Production date for several agricultural products as well as a state lookup table.    
  * United States Department of Agriculture (USDA) data portal.
  * Data was curated and provided as as CSV part (in the context) of an assignment (coursework, project)
